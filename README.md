# HOW TO USE

1. Connect a 8x8 Matrix with the following ports:
   1. Columns
        1. 1 - D8
        2. 2 - D6
        3. 3 - D4
        4. 4 - D2
        5. 5 - D11
        6. 6 - D13
        7. 7 - A1
        8. 8 - A3
   2. Rows
        1. 1 - D9
        2. 2 - D7
        3. 3 - D5
        4. 4 - D3
        5. 5 - D10
        6. 6 - D12
        7. 7 - A0
        8. 8 - A2
2. Connect a Potentiometer to the following ports
    1. Left - Voltage
    2. Center - A7
    3. Right - Ground
3. Add a knob to the top and put some tension on it with a rubberband (OPTIONAL)
4. Play the game!
